# Prompt Engineering Cheatsheet

## Prompt Formula

Role
Task
Context
Instructions
Constraints
Output Format

---

## Strong Prompt Checklist

- clear task
- defined role
- explicit input boundary
- structured output
- constraints stated
- missing info handled
- stable enough for repeated use

---

## Use Cases

### Summarization
Compress large text into:
- issue
- impact
- next step

### Inferring
Classify:
- sentiment
- urgency
- risk
- category

### Transforming
Convert:
- email to JSON
- messy text to schema
- technical text to business summary

### Expanding
Draft:
- replies
- explanations
- customer responses

---

## Golden Rule

Prompt engineering improves model behavior.
Validation and workflow control make it production-safe.
