# Project: Chatbot

## Goal

Build a simple chatbot to learn:
- role prompting
- tone control
- follow-up behavior
- prompt stability

## What to test

- short vs long responses
- concise vs detailed mode
- missing information handling
- behavior under vague prompts

## Learning outcome

A chatbot is a useful learning exercise because it teaches instruction hierarchy and conversational behavior, but it should not be confused with a governed enterprise decision system.
