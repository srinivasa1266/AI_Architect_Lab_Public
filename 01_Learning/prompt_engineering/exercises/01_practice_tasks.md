# Practice Tasks

## Exercise 1 — Improve a weak prompt

Weak prompt:
Analyze this support issue.

Task:
Rewrite it into a production-style prompt with:
- role
- task
- constraints
- output format

---

## Exercise 2 — Add structure

Prompt:
Summarize the incident.

Task:
Convert it into a JSON-returning prompt.

---

## Exercise 3 — Classification prompt

Write a prompt that classifies a customer complaint into LOW, MEDIUM, or HIGH severity.

---

## Exercise 4 — Critique prompt

Write a second prompt that reviews the recommendation from Exercise 3 and decides whether human review is required.

---

## Exercise 5 — Compare prompts

Write two versions of a summarization prompt:
- one vague
- one structured

Then compare which one would work better in a real system and why.
